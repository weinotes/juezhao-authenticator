# Contributing to JueZhao Authenticator

> Author: Davey Wong <wgwcko@gmail.com>

We love your input! We want to make contributing as easy and transparent as possible.

## Development Process

1. Fork the repo and create your branch from `main`
2. Install dependencies: `pnpm install`
3. Make your changes
4. Run type check: `pnpm type-check`
5. Run tests: `pnpm test`
6. Ensure the build passes: `pnpm build`
7. Submit a Pull Request

## Code Style

- TypeScript with strict mode
- Follow existing code style (ESLint + Prettier)
- All new features must include tests
- Use meaningful commit messages following conventional commits

## Commit Messages

```
feat(core): add SHA512 algorithm support
fix(scan): handle camera permission denied
docs(readme): update quick start
```

## Pull Request Process

1. Update the README.md with details of changes if needed
2. Update CHANGELOG.md with your changes
3. The PR will be merged once you have the sign-off

## Reporting Issues

Report bugs at [GitHub Issues](https://github.com/weinotes/juezhao-authenticator/issues).
